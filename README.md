# xuanchen-huiben

AI绘画桌面应用 - 基于Tauri + React开发

## 技术栈

- React 19 + TypeScript + Vite
- Tauri 2.x (桌面应用框架)
- Ant Design 6 (UI组件)
- Zustand (状态管理)

## 开发

```cmd
# 安装依赖
npm install

# 前端开发
npm run dev

# 构建
npm run tauri build
```

## 运行

```cmd
powershell.exe -ExecutionPolicy Bypass -File "run.ps1"
```
